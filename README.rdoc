= Phone Validator validates numbers with country code that are optionally separated by dashes.

valid:
invalid: 123, 123--123, 123x1, 123-456-7890, 1234567890

= Installation

Include the gem in your Gemfile:

  gem 'phone-validator'

  class User < ActiveRecord::Base
    validates :phone, :presence => true, :phone => true
  end
